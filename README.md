Marcelo - Starter portfolio Web3 (DeFiLlama style)
==================================================

Qué incluye
-----------
- Next.js starter (pages router)
- Simple blue (DeFiLlama-style) dark theme
- Home page, Dashboards page (Dune embed + DeFiLlama live fetch)
- Minimal dependencies (Next.js, React)

Cómo desplegar en Vercel (rápido)
--------------------------------
1. Descomprime la carpeta y subí la carpeta a un repo en GitHub (o subí directamente el ZIP en Vercel Upload).
2. En Vercel: New Project → Import Git Repository → seleccioná el repo o subí el zip.
3. Vercel detectará Next.js automáticamente. Presioná Deploy.
4. Accedé a la URL que te devuelve Vercel.

Notas
-----
- Para cambiar tu email o textos, editá pages/index.js.
- Para añadir más dashboards, editá pages/dashboards/index.js y/o crea nuevas rutas.
- Si querés, puedo generar el repo en GitHub por vos o guiarte paso a paso en el push desde PowerShell.
